###关于wing php
	wing php是一个windows平台下多进程扩展，支持守护进程、毫秒级定时器...
	php编译版本5.6.20 编译环境为Visual Studio 2012 X86 (VC11-x86)
	5.6.20下载 http://windows.php.net/downloads/releases/php-5.6.20-Win32-VC11-x86.zip

###发布的版本
	php_wing.dll 编译发布dll文件位于版本库master分支根目录
